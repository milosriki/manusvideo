## ğŸ“ Use Cases

### 1. Fitness Ad Optimization (PTD Fitness)

Analyze fitness video ads targeting Dubai men/women 40+ using proven V-Shred and WarriorBabe frameworks. Get conversion scores and actionable recommendations.

### 2. Video Content Analysis

Analyze any video content for scenes, timestamps, emotions, objects, and transcription. Perfect for content creators, marketers, and agencies.

### 3. AI Video Generation

Generate professional video ads from text prompts. Customize style, duration, and aspect ratio for different platforms.

### 4. Batch Video Processing

Process multiple videos in parallel for efficiency. Ideal for agencies managing multiple clients.

### 5. Campaign Performance Tracking

Store analysis results in HubSpot/Supabase and track performance over time. Send conversion events to Meta CAPI for ad optimization.

## ğŸš€ Getting Started

### Quick Start (5 minutes)

```bash
# 1. Clone the repository
git clone https://github.com/milosriki/manusvideo.git
cd manusvideo

# 2. Install dependencies
npm install

# 3. Configure environment
cp .env.example .env
# Edit .env and add your Gemini API key

# 4. Run development server
npm run dev

# 5. Open http://localhost:3000
```

### Production Deployment (15 minutes)

```bash
# Option 1: Google Cloud Run
gcloud builds submit --tag gcr.io/YOUR_PROJECT_ID/manusvideo
gcloud run deploy manusvideo --image gcr.io/YOUR_PROJECT_ID/manusvideo

# Option 2: Vercel
vercel --prod

# Option 3: Netlify
netlify deploy --prod
```
